================================================================================
        HOW TO BUILD YOUR APK NOW
================================================================================

Since you're on Windows without WSL, choose ONE of these methods:

================================================================================
METHOD 1: INSTALL WSL (TAKES 10 MINUTES, NO REBOOT NEEDED)
================================================================================

1. Open PowerShell as Administrator (right-click on Start menu ‚Üí PowerShell Admin)

2. Type this command and press Enter:
   wsl --install

3. Wait for installation (downloads ~500MB)

4. When asked to restart, type: Y and press Enter

5. After restart, open "Ubuntu" from Start Menu

6. Set up Ubuntu (create username and password)

7. Once in Ubuntu, run these commands:

   cd /mnt/c/Users/Hp/Desktop/myproject
   
   sudo apt update
   sudo apt install -y python3-pip python3-venv openjdk-17-jdk git zip unzip libffi-dev libssl-dev liblzma-dev zlib1g-dev
   
   python3 -m venv .venv
   source .venv/bin/activate
   pip install --upgrade pip
   pip install cython buildozer
   
   buildozer android debug

8. Wait 20-30 minutes for first build (downloads Android SDK, NDK, etc.)

9. Your APK will be in: bin/estimatepro-1.0.0-debug.apk

================================================================================
METHOD 2: GITHUB ACTIONS (CLOUD BUILD - NO LOCAL INSTALL NEEDED)
================================================================================

1. Download Git for Windows:
   https://git-scm.com/download/win

2. Install Git (use default options)

3. Open PowerShell in your project folder:
   cd C:\Users\Hp\Desktop\myproject

4. Run these commands:

   git init
   git add .
   git config user.email "you@example.com"
   git config user.name "Your Name"
   git commit -m "Initial commit"

5. Go to https://github.com/new and create a new repository

6. Copy the repository URL

7. Back in PowerShell, run:
   git remote add origin YOUR_GITHUB_URL
   git branch -M main
   git push -u origin main

8. Go to your GitHub repository ‚Üí Actions tab

9. Wait for build to complete (10-20 minutes)

10. Download APK from the Actions artifacts
